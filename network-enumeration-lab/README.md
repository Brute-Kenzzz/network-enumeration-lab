# Network Service Enumeration & Exposure Analysis Lab

## Overview
This project demonstrates network reconnaissance and service enumeration within a controlled lab environment using industry-standard tools.

## Objectives
- Identify exposed services
- Perform enumeration
- Analyze attack surface
- Document findings professionally

## Tools Used
- Nmap
- enum4linux
- Kali Linux
- FTP client

## Learning Outcomes
- Service discovery
- SMB enumeration
- FTP exposure analysis
- Network attack surface understanding
